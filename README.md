

## crud-angular-spring
application to manage completed courses


## Technology 

Here are the technologies used in this project.

* Typescript
* Angular 15 
* Java 17
* Spring



## Services Used

* Github





## Getting started

* Dependency
  - Angular Material
  - Lombok
  - Jasmine
  - Karma
 
  
  
* To install the dependencies.
```bash
  $ npm install
  ```
  
* To run the project.
    - Front-end
    - Back-end
```bash
  $ npm run start
  ```
     
  ```bash
  Rund the Spring
  
  ```
